# @customdash/ui

Composants UI reutilisables pour l'application CustomDash.

## Installation

```typescript
import { Button, Input, Card, Modal, Spinner } from '@customdash/ui';
```

## Composants

| Composant | Description |
|-----------|-------------|
| **Button** | Bouton avec variantes (primary, secondary, outline, ghost, danger) et tailles (sm, md, lg) |
| **Input** | Champ de saisie avec label, erreur et texte d'aide |
| **Card** | Carte avec sous-composants Header, Body, Footer |
| **Modal** | Fenetre modale avec Headless UI |
| **Spinner** | Indicateur de chargement anime |

## Storybook
