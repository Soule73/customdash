# @customdash/utils

Utilitaires partages pour l'application CustomDash.

## Installation

```typescript
import { formatNumber, cn, isValidEmail, getLocalStorage } from '@customdash/utils';
```

## Modules

| Module | Description |
|--------|-------------|
| **formatters** | Fonctions de formatage (nombres, devises, dates, pourcentages) |
| **helpers** | Utilitaires generaux (classes CSS, IDs, debounce, throttle, clone) |
| **validators** | Fonctions de validation (email, password, URL, empty, numeric) |
| **storage** | Helpers localStorage avec gestion d'erreurs et serialisation JSON |
